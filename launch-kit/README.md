## Usage Guidelines

### For Live Presentations
1. Start with the Cannes Line
2. Deliver the 1-minute keynote
3. Transition to live demo using the demo script
4. Leave behind the PDF

### For Async Sharing
- Send the PDF leave-behind
- Include the Cannes Line in the subject line
- Link to the platform for self-serve exploration

### For Internal Evangelism
- Use the keynote at team meetings
- Share the PDF in Slack/Teams
- Reference the voice guide when writing about the platform
